ISO/IEC 11179 objects

Below is a list of items that feature prominently within the Aristotle MetaData Registry. The following definitions are from section 3.2 of the ISO/IEC 11179-3:2013 specification. Electronic copies of the ISO/IEC 11179 specification are made freely accessible on the ISO website.

Classification Scheme

descriptive information for an arrangement or division of objects into groups based on criteria such as characteristics, which the objects have in common.

Learn more about Classification Schemes Search for Classification Schemes

Conceptual Domain

concept that expresses its description or valid instance meanings

Learn more about Conceptual Domains Search for Conceptual Domain

Data Element

unit of data that is considered in context to be indivisible

Learn more about Data Elements Search for Data Elements
Data Element Concept

concept that is an association of a property with an object class

Learn more about Data Element Concepts Search for Data Element Concepts
Data Type

set of distinct values, characterized by properties of those values and by operations on those values

Learn more about Data Type Search for Data Type
Object Class

set of ideas, abstractions or things in the real world that are identified with explicit boundaries and meaning and whose properties and behaviour follow the same rules

Learn more about Object Classes Search for Object Classes
Property

quality common to all members of an object class

Learn more about X Search for Properties
Value Domain

set of permissible values
